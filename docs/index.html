Physical AI Textbook

Learn Robotics: From ROS 2 to Robot Control

An interactive, hands-on introduction to building and controlling robots using ROS 2 Humble, Python, and simulation

Welcome​

This is a comprehensive, free textbook for learning robotics from the ground up. Whether you're a student, hobbyist, or professional engineer, you'll find practical, in-depth tutorials covering the complete pipeline from understanding ROS 2 to controlling real robots.

What You'll Learn​

Core Skills​

  • How ROS 2 works and how robots communicate
  • Describe robot structure using URDF
  • Control robots with Python code
  • Visualize and simulate robots

Practical Abilities​

  • Launch and manage ROS 2 nodes
  • Publish velocity commands to move robots
  • Configure behavior using parameters
  • Implement goal-seeking actions

Real-World Applications​

  • Mobile robot navigation
  • Robot arm manipulation
  • Multi-robot coordination
  • Autonomous behaviors

Course Structure​

Module 1: ROS 2 Fundamentals (3-4 hours)​

The complete foundation for robotics development in 3 chapters:

Chapter 1: ROS 2 Basics (40 min)

  • Nodes, topics, and services
  • The ROS 2 graph
  • Communication patterns

Start Chapter 1

Chapter 2: URDF Robot Description (40 min)

  • Robot structure with links and joints
  • Physics properties and simulation
  • Visualization in RViz and Gazebo

Start Chapter 2

Chapter 3: Python Integration (40 min)

  • Writing Python ROS 2 nodes
  • Publishing velocity commands
  • Using parameters and actions

Start Chapter 3

How to Use This Textbook​

For Self-Study​

  1. Start with Module 1
  2. Read each chapter sequentially
  3. Work through all code examples
  4. Complete the exercises
  5. Review summaries and mental models

For Classroom​

  • Use chapters as lecture materials
  • Assign exercises as homework
  • Use code examples for demonstrations
  • Reference summaries for key concepts

For Hands-On Learning​

  • Type out all code (don't copy-paste)
  • Modify examples and experiment
  • Complete exercises with your own code
  • Build projects using learned skills

Key Features​

Progressive Difficulty​

  • Beginner: Understand fundamentals
  • Intermediate: Apply concepts to real problems
  • Advanced: Design and optimize solutions

Practical Code Examples​

  • All code is tested and documented
  • Detailed comments explain concepts
  • Ready to run in ROS 2 Humble
  • Available for reference or modification

Hands-On Exercises​

  • 3 exercises per chapter (9 total)
  • Progressive difficulty levels
  • Complete solutions provided
  • Success criteria clearly stated

Mental Models​

  • Conceptual diagrams
  • Analogies and comparisons
  • Common mistakes explained
  • Why things work (not just how)

Requirements​

Knowledge​

  • Programming: Basic Python experience
  • Linux/Terminal: Comfortable with command line
  • Git: Basic clone and commit

Software​

  • ROS 2 Humble installed
  • Gazebo simulator
  • Python 3.10+
  • Text editor (VS Code recommended)

Hardware​

  • Any modern computer (Linux/WSL recommended)
  • No specific robotics hardware required
  • All examples use simulation

Success Criteria​

After Module 1, you'll be able to:

✅ Explain ROS 2 architecture ✅ Use ROS 2 command-line tools
✅ Write Python ROS 2 nodes ✅ Create and visualize URDF robots ✅ Control robots with velocity commands ✅ Configure behavior with parameters ✅ Implement goal-reaching actions ✅ Debug common robotics issues
