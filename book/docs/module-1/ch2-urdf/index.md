## What You'll Learn

This chapter teaches you how to describe robot structure using URDF (Unified Robot Description Format).

### Topics Covered

- URDF basics (links and joints)
- Physical properties (mass, inertia)
- Collision and visual geometry
- Robot visualization in RViz
- Simulation in Gazebo

### Learning Objectives

By the end of this chapter, you will be able to:
- Define robot structure using URDF
- Create valid URDF files
- Visualize robots in RViz
- Debug URDF syntax errors
